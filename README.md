# MP3-Tag-Reader-and-Editor
MP3 Tag Reader implemented in C that reads and displays MP3 metadata (ID3 tags) 

such as title, artist, album, and year from audio files. The command-line application parses binary 

MP3 data using file handling and string operations to extract and display tag information efficiently.


## 📌 Project Overview
• MP3 Tag Reader is a command-line application developed in C to read and display MP3 metadata information.

• The program extracts ID3 tag details such as title, artist, album, year, and genre from MP3 files.

• It reads binary data from the audio file and parses the metadata fields stored in the ID3 tag structure.

• The application helps users quickly view MP3 file information without using any external media software.

---

## ✨ Features
• Read MP3 (ID3) tag information  
• Display metadata such as Title, Artist, Album, Year and Genre  
• Detect ID3 tag version used in the MP3 file  
• Display help menu using command line options  
• Handle error messages if tag information is not found  

---

## 🛠 Technologies Used
• C Programming  
• File Handling (Binary File I/O)  
• String Manipulation  
• Pointers and Functions  
• Command Line Arguments  

---

## ▶️ Execution
Run the program from the command line by providing an MP3 file.

./mp3_tag_reader song.mp3
